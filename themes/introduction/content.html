Introduction to Machine Learning

What is Machine Learning?

Artificial Intelligence > Machine Learning > Deep Learning

  • Artificial Intelligence: Machines that mimic human Intelligence and cognitive functions.
  • Machine Learning: Algorithms trained with data to make preditions.
  • Deep Learning: Subset of Machine Learning with automated feature extraction.

How is the learning process?

  1. Business Problem: Understand which problem I want to solve doing things like talk with stakeholders.
  2. DataAnalysis: A deep dive into the available data to understand what we can learn from that and even where can we go with it.
  3. Data preparation: Is the procces of handling and cleaning the data.
  4. Modeling: Is the creation of the Machine Learning model with the prepared data.
  5. Evaluation: Evaluation of the Machine Learning model using metrics.
  6. Deploy: Deploy the Machine Learning model to productive environment, using it to take decisions.

Types of learning

  • Supervised: I teach my model what was righ and wrong in the past, and it try to reproduce my decisions with new data.
  • Unsupervised: I give a sample of data and the model learns by itself splitting the data into groups based on how they are similiar to each other.
  • Reinforcement: I gave rules of a universe to the model iteract and it evolves based on trial and error.

Supervised Learning

In this type of learning, the model make predictions using data from the past.

Classification

The objective is to predict from which class an new event is using data from previous events that we know their classifications.

To make predictions I use many features to create a mathematical formula that can explain (or be near to explain) this relationship (between de features [x] and the target [y])

Example: I want to know if somebody have a decease or not using historical data of people that has and hasn't the decease.

Regression

It's similar to the Classification but the objective here is to predict the value of an event.

The mathematical idea of this method is to create a function that explains the behaviour of the problem, using two coeficients α(intercept) and β(slope).

Example: I want to know how much does this house costs.

Unsupervised Learning

In this type of learning, we don't know the outputs, so I give only the inputs to the model. The model is responsible to find patterns and cluster the data based on how they are similiar.

Examples of clustering models:

  • K-means.
  • Singular Value Decomposition (SVD).
  • Principal Component Analysis(PCA).

Reinforcement Learning

In this type of learning, the model learns based on trial and error. It is used in robotics and game development.

For each iteraction of the model and the environment, the model receives a reward and learns with it. It's similiar to train dogs to make tricks.

How tho choose the correct Algorithm?

It depends of the problem. I need to know the context like which business problem is, how much time I have, how much money I have to use.

Six factors to choose the Algorithm

  1. Precision: How precise is the model. There is a trade of, a very precise model can be overfitted and do not be applied to real data.
  2. Time to train: Some models can be very slow to train
  3. Resources: How much resources can I use.
  4. Type of problem: What is the question I'm looking for answer.
  5. Linearity: If there is any linearity that can explain the behaviour of this phenomenon.
  6. Number of parameters: Some Algorithms can become very heavy with many algorithms.
