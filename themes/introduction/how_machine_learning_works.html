How machine learning work?

What is the difference between Machine Learning and Deep Learning?

While Machine Learning you have to make the feature extraction (process to turn multidimensional data into unidimensional features), in Deep Learning it is made by the own model.

The biggest problem of Deep Learning is the biggest need for resources and time to run.

Fundamental elements for Machine Learning

  • Pattern: The Machine Learning is only able to learn problems with patterns.
  • There is not only one mathematical model that solve that problem: If the problem is simple and we know exactly which equation solves it, why would I want to use an Machine Learning model to calculate an "aproximate" equation?.
  • Available data: If you don't have data stored then there is no way you can use Machine Learning.

Components of the learning proccess

  • Input (x): Features of each entry.
  • Output (y): Is the target I want to predict.
  • Target function (f:x -> y): Unknown mathematical equation.
  • Data (x1,y1): Historical data.
  • Hipotesis (g:x -> y): Aproximation of the unknown mathematical equation that my model could create.

What is the difference between the Hypothesis Space and the Learning Algorithm?

The Hypothesis Space contains the resources we can use to solve this problem (Like Artificial Neural Network or SVM)

The Learning Algorithm is, looking inside the Hypothesis Space, how I will handle this problem (Like Back Propagation and Quadractic Programation).
